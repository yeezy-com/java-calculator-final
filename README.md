# java-calculator-final
### 최종 코테 대비..

입력 문자열에서 숫자를 추출하여 더하는 계산기.
- 쉼표, 콜론을 구분자로 가지는 문자열
  - 구분자를 기준으로 분리하여 각 숫자의 합 반환
- 기본 구분자 외 커스텀 구분자 지정
  - 문자열 앞 부분의 "//", "\n" 사이의 "문자" 커스텀 구분자로 사용
- 잘못된 값 입력
  - `IllegalArgumentException` 발생 후 종료

- 구분자
  - 기본 구분자
    - 쉼표, 콜론
  - 커스텀 구분자
    - "문자열 앞 부분" //, \n 사이의 "문자"
      - 하나의 커스텀 구분자만 허용

생각해볼 수 있는 객체
1. 계산기
2. 구분자
3. 숫자
